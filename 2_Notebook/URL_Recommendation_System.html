This data set consists of 99,983 popular URLs bookmarked on Delicious within a past time window. Each URL includes the date first saved, the number of saves, and the top 10 tags used and their respective counts.

The time window for this dataset is defined by the date of first save for the popular URLs. The oldest "first save date" is from late 2006, but almost every one of the indicated URLs has had save activity since that date.

The file is tab delimited; the columns are (from left to right):

  • URL
  • Number of saves
  • Date of first save
  • Tag
  • Tag count
  • [last two repeat to represent the top 10 tags and their counts]

def recommend(url, df):
    if url not in df['URL'].values:
        return f"Error: URL '{url}' not found in the dataset."
    
    idx = df.index[df['URL'] == url][0]
    distances, indices = model.kneighbors(tfidf_matrix[idx])
    recommended_indices = indices[0][1:]
    return df.iloc[recommended_indices]['URL'].values
